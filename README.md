PuzzleYou is an interactive mobile puzzle game that turns your world into a playground.By capturing images of your surroundings,
the app dynamically generates a 4x4 grid of tiles that players must drag and rotate to complete the original image.

🚀 Key Features

Dynamic Image Capture: Uses the device camera to convert real-time photos into a 4x4 puzzle grid.
Interactive Mechanics: Features intuitive drag-and-drop controls combined with tile rotation logic.
Native Performance: Developed using [Java/Kotlin] for optimal performance and smooth UI interactions.
User-Centric UI: A clean interface designed to turn any environment into a challenging game.

🛠 Tech Stack

Platform: Android (Native)
Language: [Java]

💡 How it Works

Take a photo of your surroundings.
The app fragments the image into a 4x4 grid.
Randomize tiles and challenge yourself to drag and rotate them back into place!
